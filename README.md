# 散修崛起：斬妖煉天訣 🗡️

一款**單機 2.5D 等角動作修仙 RPG**，純前端、單一 HTML 檔、手機可玩。
你是一介無門無派的散修，靠打坐吐納、揮劍斬妖，一路突破境界，登仙路自己殺出來。

> 這是 Riley 的個人練習專案，用「多 AI agent 協作」方式開發（編劇 / 遊戲設計 / 程式 / 測試各司其職）。
> 開發路線採分階段：**Phase 1 先用 2.5D 出可玩 MVP（本版本）→ Phase 2 再升級為 Three.js 真 3D。**

---

## 🎮 怎麼玩

最簡單：用瀏覽器（Chrome / Edge / 手機瀏覽器）直接打開 **`index.html`** 即可遊玩，免安裝、免連網。

- 桌機：`WASD` / 方向鍵移動，`J` 普攻，`Q / E / R` 放技能，滑鼠也可操作介面。
- 手機：左下**虛擬搖桿**走位，右下**攻擊鈕 + 技能鈕**，底部分頁切換各功能。

## ✨ 玩法系統

| 系統 | 說明 |
|------|------|
| 🧘 放置打坐 | 進入打坐狀態，靈氣隨時間自動累積；支援離線收益（上限 8 小時、60% 效率）|
| ⚡ 突破境界 | 靈氣達標可突破，有成功率，失敗損失部分靈氣（煉氣 → 散仙共 10 境）|
| ⚔️ 等角戰鬥 | 斜 45° 俯視即時動作，走位斬妖、放技能，擊殺得經驗與靈石 |
| 🎲 隨機奇遇 | 外出歷練觸發好 / 壞 / 中性事件，影響靈氣、靈石、屬性 |
| 🔥 煉丹房 | 依境界解鎖丹方，煉製需時間，開出 5 種隨機稀有度（凡品 → 仙品）丹藥 |
| 🪙 每日拍賣 | 付靈石入場，拍賣官主持，NPC 智能喊價（達隱藏上限即退出，不坑玩家），價高者得 |

## 🛠️ 技術

- 單一 `index.html`，原生 **HTML5 Canvas + JavaScript**，無任何外部依賴（無 CDN / 無框架）。
- `requestAnimationFrame` 遊戲迴圈，`localStorage` 自動存檔。
- 響應式版面，同時支援手機觸控與桌機鍵鼠。
